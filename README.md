# IP Logger
Log website visitor information with PHP

## Functionality
When an user visits the `index.php` page, their information gets logged into `log.txt`. The information can be viewed in a HTML table form by visiting the `log.php` page.

## Usage notices
The program logs information such as IP addresses. Therefore, the log files should be protected from public visitors. Also, it is encouraged to inform any website visitors for possible information logging. This could be enforced by law in some places.
